Micro League Baseball

MLBB was one of the first video games to carry the Major League Baseball license, allowing the game to feature MLB teams. It also carried the Major League Baseball Players Association license, allowing the game to use real players.

Options

N
- to begin a New game
D
- for a Demo of the game
R
- to Restart a partially completed game

Starting a Game

Pick the visiting and home team and hit the RETURN Key if you want to manage the team or hit C to have the computer manage the team. Next, select starting pitchers and manage batting lineups and fielding positions. Hit ESC and the game is ready to begin.
